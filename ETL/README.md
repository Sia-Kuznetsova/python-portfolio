# Проект: ETL-парсинг товарного справочника: извлечение атрибутов из неструктурированных данных

## Задача
Исходные данные представляли собой «грязный» CSV-файл с информацией о товарах, где все атрибуты (код товара, бренд, название, размер, цвет)
находились в одной строке в неструктурированном виде.
Требовалось разбить эти данные на отдельные логические колонки для дальнейшего анализа.

**Исходные данные** выглядели так:
PFAN-1107, Döcke Наличник 89 мм/ 30 мм (Белый)

**Результат** — аккуратный DataFrame с 8 логическими колонками:

| Code | Brand | Name | Size | Color |
|------|-------|------|------|-------|
| PFAN-1107 | Döcke | Наличник | 89 мм/ 30 мм | Белый |

## Цель
Разбить неструктурированную строку из колонки `Код` на отдельные атрибуты:
- Код товара (Code)
- Бренд (Brand)
- Название товара (Name)
- Размер (Size)
- Цвет (Color)
- Количество в упаковке (quantity)
- Объём на единицу товара (m3_per_one)
- Вес брутто на единицу (gross_per_one)

## Технологии

| Технология | Назначение |

| **Pandas** | Работа с DataFrame, ETL-операции |
| **NumPy** | Математические вычисления |
