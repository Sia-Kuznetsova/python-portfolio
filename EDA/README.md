# Проект: Исследовательский анализ данных (EDA) рынка недвижимости

## Описание проекта

В этом проекте я провожу исследовательский анализ данных (EDA) для датасета с информацией о продажах домов в округе Кинг, штат Вашингтон (США).
Датасет содержит 21 признак и более 20 тысяч записей.

**Цель проекта** — изучить структуру данных, выявить закономерности и определить ключевые факторы, влияющие на стоимость недвижимости. 
Такой анализ нужен для:
- оценки рыночной ситуации
- построения моделей ценообразования
- принятия инвестиционных решений

## Данные

**Источник:** [Kaggle — House Price Dataset with Other Information](https://www.kaggle.com/datasets/esratmaria/house-price-dataset-with-other-information)

**Описание основных полей:**

| Поле | Описание |
|------|----------|
| `price` | Цена дома (целевая переменная) |
| `bedrooms` | Количество спален |
| `bathrooms` | Количество ванных комнат |
| `sqft_living` | Жилая площадь (кв. футы) |
| `sqft_lot` | Площадь участка (кв. футы) |
| `floors` | Количество этажей |
| `waterfront` | Наличие вида на набережную (0/1) |
| `view` | Качество вида (0-4) |
| `condition` | Состояние дома (1-5) |
| `grade` | Общий класс качества дома (1-13) |
| `yr_built` | Год постройки |
| `yr_renovated` | Год реновации |

## Задачи исследования

1. **Изучить распределение ключевых метрик:**
   - стоимости недвижимости
   - жилой площади
   - года постройки

2. **Проанализировать категориальные признаки:**
   - влияние наличия вида на набережную
   - распределение этажности
   - распределение состояния домов

3. **Выявить факторы, влияющие на стоимость:**
   - построить не менее 5 диаграмм для визуализации взаимосвязей
   - определить корреляции между признаками
   - сформулировать выводы

## Используемые инструменты

- **pandas** — загрузка, очистка, агрегация данных
- **numpy** — численные расчеты
- **matplotlib / seaborn** — визуализация
- **scipy** — статистические тесты
